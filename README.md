# RedisSimulator
this is app to simulate redis with python 

command help :

    SET :
        SET key value

        SET key value --ttl 10 

        SET key value1,value2,value3 

        SET key value1,value2,value3,value2 --set True 

    GET :
        GET key
    
    DEL :
        DEL key

    TTL :
        TTL key

    ALL :
        ALL
